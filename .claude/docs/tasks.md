### M5 关键结论

- **测试集统一重评估**：10 个模型在测试集上指标与 M4 报告（验证集）差异显著，MLP f1 从 0.989→0.720
- **深度学习测试集表现与 DT 相近**（f1≈0.72），M4 报告的高指标源自训练集验证划分
- **多分类仍受限于 unseen attacks**：MLP full_acc=0.101 为最佳，RF=0.046
- **SMOTE 失败确认**：full_acc 从 0.101→0.028（-72%）
- **7 张对比图表 + 9 章节报告**全部产出，新增 14 个评估测试
- **3 次原子提交**（evaluation → orchestration → report）

### M4 关键结论

- **二分类 MLP 显著优于 RF**（f1: 0.989 vs 0.688, +30%; auc: 0.999 vs 0.915, +8%）
- **多分类 MLP 略优于 RF**（full_acc: 0.101 vs 0.046, +117%，但仍受 unseen 攻击拖累）
- **SMOTE 在本项目中失败**（full_acc -72%），副作用已诚实记录（O-NN-04）
- **tabular CNN/LSTM 略低于 MLP**（论文扩展性验证维度）
- **44 个 M4 新测试全部通过**（含 grid search / CNN / LSTM / SMOTE / torch 持久化）
